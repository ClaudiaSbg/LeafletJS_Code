Einleitung

Wenn man Karen erstellen will, ohne auf Services wie Google Maps angewiesen zu sein kann man dafür z.B. Leaflet JS nutzen und auf Basis dessen und Services wie Open Streetmal und offenen Geodaten selber Karten erstellen.

Man benötigt dafür folgendes:

  • Ein Programm zum Bearbeiten von Texten z.B.: Notepad++
  • Informationen zu Leaflet z.B.: Leaflet Website
  • Idealerweise einen Server oder Webspace zum Raufladen der Dateien
  • Falls man die Dateien auf einen Webspace laden will braucht man noch ein FTP bzw. SFT Programm z.B.: Filezilla (Client)

Leaflet basiert auf

  • Javascript => Interaktivität und Struktur
  • CSS (Cascading Style Sheets) => Design
  • HTML (Hypertext Markup Language => Rahmen

Rahmen

Zum Start öffnet man sein Programm zum Texte bearbeiten und speichert die Datei mit der Endung html ab z.B.: karte.html

Dann muß man einen Rahmencode reinkopiern

        <!DOCTYPE html>
        <html>
        <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Karte</title>
        </head>
        <body>
        </body>
        </html>

Man muß dann den Code von Leaflet einbinden. Man kann ihn auf den eigenen Server/ Webspace laden aber man kann ihn auch online einbinden. Er mu&szli; in den Header eingebunden werden, also zischen die Header Tags.

CSS

        <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" 
        integrity="sha256-p4NxAoJBhIIN+hmNHrzRCf9tD/miZyoHS5obTRR9BMY=" 
        crossorigin=""/>

Javascript

        <!-- Make sure you put this AFTER Leaflet's CSS -->
        <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js" 
        integrity="sha256-20nQCchB9co0qIjJZRGuk2/Z9VM+kNiyxNV1lvTlZBo="  
        crossorigin=""></script>

Man braucht dann noch einen Bereich, in dem die Karte angezeigt wird. Dieser muß im body der Seite sein, also zwischen den body Tags.

        <div id="map"></div>

Und man muß den Bereich auch noch Formtieren, zumindest die Breite. Das macht man mit CSS.

        <style>
            #map { height: 600px; }
        </style>

Zusammen schaut es dann so aus:

    <!DOCTYPE html>
    <html>
    <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Karte</title>

    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" 
    integrity="sha256-p4NxAoJBhIIN+hmNHrzRCf9tD/miZyoHS5obTRR9BMY=" 
    crossorigin=""/>

    <!-- Make sure you put this AFTER Leaflet's CSS -->
    <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js" 
    integrity="sha256-20nQCchB9co0qIjJZRGuk2/Z9VM+kNiyxNV1lvTlZBo="  
    crossorigin=""></script>

    <style>
        #map { height: 600px; }
    </style>
    </head>

    <body>
    <div id="map"></div>

    </body>
    </html>

Erste Karte

Der Code für die Struktur einer Karte ist Javascript. Dieser wird üblicherwise in script Tags in den Body Tag kopiert. Der Folgende Code

  • Definiert eine Karte und zeigt sie im Bereich map <div id="map"></div> an
  • Die Zahlen in Eckigen Klammern sind die Koordinaten des Mittelpunkts der Karte.
  • Die nächste Zahl der Zoomfaktor
let map = L.map('map').setView([47.8,13.033333], 13);

Der Code würde eine "leere Karte2 anzeigen. Man könnte eine eigene Basiskarte verwenden, aber man kann auch OpenStreetmap einbinden

       L.tileLayer('https://tile.openstreetmap.org/{z}/{x}/{y}.png', {
            maxZoom: 19,
            attribution: '© <a href="http://www.openstreetmap.org/copyright">OpenStreetMap</a>'
       }).addTo(map);

Punkt setzen

Damit mehr zu sehen ist kann man aber auch noch einen Marker setzen. Dazu kopiert man folgendes in den Script Tag unter den Tile Layer Code

        var marker = L.marker([47.788698035838806, 13.060707859689483]).addTo(map);

Die Koordinaten kann man natürlich anpassen. Wichtig ist nur, dass sie innerhalb des gewählten Kartenausschnitt sind.

Man kann dem Marker auch ein Popup hinzufügen:

        marker.bindPopup("Nawi").openPopup();

Zusammen schaut das dann so aus:

    crossorigin=""></script>

    <style>
